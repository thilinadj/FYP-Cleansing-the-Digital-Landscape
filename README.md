# Command list
## virtual environment
### Initiate the virtual environment
```
python -m venv .venv
```

### Activate the virtual environment
#### bash command:
```bash
source .venv/Scripts/activate
```
#### cmd command:
```cmd
.\.venv\Scripts\activate
```
### Install dependencies
```bash or cmd
pip install -r requirements.txt
```

### Deactivate the venv
```bash or cmd
deactivate
```
